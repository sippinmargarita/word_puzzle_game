# Word Puzzle Game
A small Python word puzzle game built with a university-provided library. Players choose a difficulty (easy, medium, hard) and must correctly type one of two displayed words before time runs out. Earn points for perfect entries, lose hearts for mistakes, and win by reaching the score goal.

![Game Demo](resources/word_puzzle_demo.gif)

# Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sippinmargarita/word_puzzle_game.git
   ```
2. Navigate into the project folder:
    ```bash
   cd word_puzzle_game
    ```
3. Run the game:
    ```bash
   python project.py
   ```
